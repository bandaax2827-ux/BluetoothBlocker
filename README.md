# BluetoothBlocker-1.6.3
Блокировщик Bluetooth, если он мешает вам или сам включается. Мы обновили приложение, и теперь оно поддерживается буквально на всех Android телефонах, включая Samsung, OnePlus, Huawei, Realme и Xiaomi.
  Безопасное удаление позволяет удалить Bluetooth Blocker без вреда для системы.
